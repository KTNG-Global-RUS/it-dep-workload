Настройка общего доступа через Firebase

После настройки файл можно положить на общий диск или открывать по ссылке — данные будут автоматически синхронизироваться между всеми пользователями в реальном времени.
1
Создайте проект Firebase

Перейдите на console.firebase.google.com, войдите через Google-аккаунт и нажмите «Добавить проект». Google Analytics можно отключить.

2
Создайте базу данных Realtime Database

В левом меню: «Создание» → «Realtime Database» → «Создать базу данных». Выберите регион (europe-west1 для РФ). Выберите «Начать в тестовом режиме» (правила на чтение/запись открыты).

3
Получите конфигурацию

В настройках проекта (⚙ → «Настройки проекта») → вкладка «Общие» → прокрутите вниз → «Ваши приложения» → нажмите </> (веб), зарегистрируйте приложение. Скопируйте объект firebaseConfig.

4
Вставьте конфиг в файл

Откройте этот HTML-файл в текстовом редакторе (Notepad++ / VS Code). Найдите строку:

const FIREBASE_CONFIG = null;

Замените на ваш конфиг:

const FIREBASE_CONFIG = { apiKey: "AIzaSy...", authDomain: "ваш-проект.firebaseapp.com", databaseURL: "https://ваш-проект-default-rtdb .europe-west1.firebasedatabase.app", projectId: "ваш-проект", storageBucket: "ваш-проект.appspot.com", messagingSenderId: "123456789", appId: "1:123...:web:abc..." };
5
Разместите файл и откройте с любого ПК

Положите файл на общий сетевой диск или в Google Drive / OneDrive. Каждый пользователь открывает его в браузере — все изменения мгновенно синхронизируются.

⚠ Тестовый режим Firebase открывает доступ к базе для всех. Для корпоративного использования настройте правила безопасности в консоли Firebase (вкладка «Правила»).
