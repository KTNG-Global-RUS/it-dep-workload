Настройка общего доступа через Firebase

Файл уже содержит Firebase-конфигурацию. При корректных правилах Realtime Database изменения синхронизируются между пользователями.
1
Проверьте Realtime Database

Откройте Firebase Console → Realtime Database → Data и убедитесь, что база создана.

2
Проверьте URL
https://it-dep-workload-default-rtdb.europe-west1.firebasedatabase.app
3
Настройте безопасность

Не оставляйте тестовые правила открытыми для публичного доступа в корпоративной эксплуатации.

⚠ API-ключ Firebase не является секретом, однако доступ к данным должен ограничиваться правилами базы и аутентификацией.
